## About
This is an app that allows coelacanth to swim in the desktop screen of Windows10.  
It appears below all windows and icons so it doesn't get in the way of your work.  
Coelacanth does nothing special. Just swims.  
<br>
<img src="https://github.com/hebinosuke/DesktopCoelacanth/blob/images/images/screenshot.png" width="512">
<img src="https://github.com/hebinosuke/DesktopCoelacanth/blob/images/images/background.png" width="512">

## Installation
It can be installed from Microsoft Store(free).
  
## How to Use
* **Run**  
Press windows key and search for `DesktopCoelacanth`.  
Due to PC performance, it may take a few minutes to run.

* **Settings and Exit**  
Find DesktopCoelacanth in the System Tray Icons on the taskbar and **right click** on the icon.<br>
<img src="https://github.com/hebinosuke/DesktopCoelacanth/blob/images/images/systemtray.png"><br>
  * **Swimming Mode**  
    1. `Manual` : Coelacanth follows the mouse cursor.  
    2. `Auto` : Coelacanth swims randomly.
  * **Monitor**  
  You can select the monitor to display.  
  You will be prompted to restart after selection.
  * **Screen Quality**  
  You can select the quality of the screen display.  
  The better the quality, the more resources it consumes(mainly GPU).  
  You will be prompted to restart after selection.
  * **Background**  
  Select `Slideshow` if you have set your wallpaper as a slideshow, or want to choose a wallpaper while running the app.  
  Otherwise select `Picture`(saves GPU).
  * **Restart**  
  Restart the app.
  * **Exit**  
  Quit the app.

## Run at Windows Startup
1. Press `Win + R` and enter `shell:appsfolder`.
2. Find DesktopCoelacanth and select `right-click`(maybe + Shift key) > `Create shortcut`.
3. Press `Win + R` again and enter `shell:startup`.
4. Move the shortcut to the displayed folder.

## Uninstall
Press windows key and search for `DesktopCoelacanth` and select `right-click` > `Uninstall`.  
If you don't need it, you can delete the temporary folder in the following path.  
`C:\Users\[UserName]\AppData\LocalLow\DesktopCoelacanthProject`

## Privacy Policy
1. This app just runs the exe file inside this package and saves the configuration file, temporary files and logs in the user's AppData folder.
2. This app does not collect user's personal information.
3. This app does not collect user activity.
4. This app does not send any information to the outside.

## Precautions
1. We are not responsible for any damage caused by this product.
2. It may not work properly in S mode of Windows10.
3. If the wallpaper does not recover when finished, press `Win + Tab` to update the wallpaper.
4. Email to `kokume01@gmail.com` for inquiries.

# DesktopCoelacanth(日本語)

## 概要
このアプリは、シーラカンスをWindows10のデスクトップ画面で泳がすことができます。  
すべてのウィンドウとアイコンの下に表示されるため、作業の邪魔になりません。  
シーラカンスは特別なことをしません。ただ泳ぐだけです。  
<br>
<img src="https://github.com/hebinosuke/DesktopCoelacanth/blob/images/images/screenshot.png" width="512">
<img src="https://github.com/hebinosuke/DesktopCoelacanth/blob/images/images/background.png" width="512">

## インストール
Microsoftストアから無料でインストールできます。
  
## 使い方
* **起動**  
Windowsキーを押して、`DesktopCoelacanth`を検索して実行します。  
PCのパフォーマンスにより、起動までに数分かかる場合があります。

* **設定・終了方法**  
タスクバーのシステムトレイでDesktopCoelacanthのアイコンを見つけ、アイコンを**右クリック**します。<br>
<img src="https://github.com/hebinosuke/DesktopCoelacanth/blob/images/images/systemtray.png"><br>
  * **Swimming Mode**  
    1. `Manual` : シーラカンスはマウスカーソルに追従します。  
    2. `Auto` : シーラカンスはランダムに泳ぎます。
  * **Monitor**  
  表示するモニターを選択できます。  
  選択後、アプリの再起動を求められます。
  * **Screen Quality**  
  画面表示の品質を選択できます。  
  品質が高いほど、より多くのリソース（主にGPU）を消費します。  
  選択後、アプリの再起動を求められます。
  * **Background**  
  壁紙をスライドショーとして設定した場合、またはアプリの実行中に壁紙を選びたい場合は、`Slideshow`を選択します。  
  そのほかの場合は`Picture`を選択します（GPU消費を節約できます）。
  * **Restart**  
  アプリを再起動します。
  * **Exit**  
  アプリを終了します。

## スタートアップに設定
1. `Win + R`を押し、`shell:appsfolder`と入力して実行します。
2. DesktopCoelacanthを探し、`右クリック`(+ Shiftキー) > `ショートカットの作成`を選択します。
3. 再度`Win + R`を押し、`shell:startup`と入力して実行します。
4. 表示されるフォルダに、先程のデスクトップに作成されたショートカットを移動します。

## アンインストール
Windowsキーを押して、`DesktopCoelacanth`を検索し、`右クリック` > `アンインストール`を選択します。  
不要な場合は、次のパスの一時フォルダを削除しても構いません。  
`C:\Users\[UserName]\AppData\LocalLow\DesktopCoelacanthProject`

## プライバシーポリシー
1. このアプリはパッケージ内のexeファイルを実行し、構成ファイル、一時ファイル、およびログをユーザーのAppDataフォルダに保存します。
2. このアプリはユーザーの個人情報を収集しません。
3. このアプリはユーザーの活動を収集しません。
4. このアプリは外部に情報を送信しません。

## 注意事項
1. 本製品によって生じた如何なる損害については責任を負いかねます。
2. Windows10のSモードでは正常に動作しない場合があります。
3. 終了しても壁紙が復元しない場合は、`Win + Tab`を押して壁紙を更新してください。
4. お問い合わせは`kokume01@gmail.com`までメールをお願いします。
